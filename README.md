# DataMaster · AI 经营分析工作台

> Local-first business analysis: drop in Excel/CSV, get evidence-backed insights.

DataMaster 是一个**本地优先**的个人经营分析桌面应用（Windows / macOS）。把日常经营中零散的 Excel、XLS、CSV 报表交给它，就能在本机自动完成合并、质量检查与字段识别，输出五个视角的经营分析、可复核的证据账本，以及 Excel 分析表、Word 经营报告和 PDF 详细报告。**全部计算在本机完成，经营数据不出本机。**

## 它解决什么问题

- 每个月的销售明细散落在多个 Excel 里，人工合并容易重复计数、对错口径。
- 同一张表里常有未税金额、含税金额、调后金额、多套成本列，算错一列利润全错。
- 想知道"哪个产品赚钱、哪个客户亏钱、哪个销售组在拖后腿"，但透视表做一次要花半天。
- 想用 AI 帮忙解读，又不愿意把经营数据上传到任何云端。

## 功能详情

### 1. 多源导入与智能合并

- 一次拖入多个 `.xlsx` / `.xls` / `.csv`，或分批追加；单文件最高 500 MiB，超大文件自动 SAX 流式解析。
- 自动跳过名称明显为"汇总/总览"的工作表，避免明细与汇总重复计数。
- 多文件按统一口径自动合并成一份明细，并给出数据质量报告（缺日期、缺产品、数值异常等）。

### 2. 字段自动识别与口径确认

- 按中英文同义词自动映射：日期、产品、客户、收入、成本、费用、数量、出库数量、换算只数、调拨成本、销售区域、销售部门、销售组、渠道、规格型号、送货客户、送货地址，以及一~六级分类、产品形态、仓库、库存组织、等级等维度。
- 检测到**多套口径候选**（如未税/含税/调后收入、多列成本）时，程序**不会静默猜选**：弹出字段口径卡，由你确认后才会计算，原始文件不会被修改。
- 不确定怎么选时，可使用内嵌的 **AI 辅助识别字段**（只发送匿名列画像，不发送原始数据）。

### 3. 五个分析视图

| 视图 | 内容 |
| --- | --- |
| 经营总览 | 收入/利润 KPI、月度收入成本利润趋势、关键驱动因素、数据质量分 |
| 产品分析 | 产品收入与利润率、结构透镜（多口径占比环图）、量价拆解、明细搜索与多选筛选 |
| 客户分析 | 客户集中度、价值明细（冻结表头长表）、亏损客户清单（含销售组）、客户→产品利润多选合并 |
| 利润与成本 | 百元收入结构、扭亏路径、毛利率趋势、**动态维度浏览器**（20+ 业务维度任意切换） |
| 销售组利润 | 阿米巴口径：调拨成本/已确认成本双口径对照、区域/部门/销售组三层筛选、客户与产品下钻 |

### 4. 数据驱动的维度下钻

程序会**自动从数据中发现层级关系**并生成下钻入口，而不是写死规则：某个维度的值几乎都唯一归属于另一维度、且能展开成更细粒度时，就开放"父 → 子"下钻。例如本套数据发现了 部门 → 区域 → 销售组、送货客户 → 送货地址、产品形态 → 规格型号、库存组织 → 仓库 等 20 余条路径；换一套表格，它会给出那套表格自己的下钻路径。支持下钻后一键返回，子级还可以继续下钻。

### 5. 产品规格与表格体验

- 有"规格型号"列时，产品名后自动挂规格徽章；一个产品存在多种规格时标注"等 N 种"，鼠标悬浮列出全部变体。
- 长表格统一冻结首行表头；表格滚到底后滚动自然交接给页面，不卡手。
- 金额数字统一等宽数字字形对齐；表格行悬浮高亮；深色 tooltip 即时跟随。

### 6. 证据账本

每条结论都挂证据编号（EV-XXX），点击即可打开证据抽屉，回到**确定性计算**的口径、来源与数值复核。AI 生成的建议与确定性计算严格隔离，不会改写任何数字。

### 7. AI 辅助分析（可选）

- 支持 **DeepSeek** 与**阿里云百炼**：按"平台 → 模型"两级配置，支持官方预设与自定义模型 ID；API Key 经 AES-GCM 加密保存在本机且不回显。
- 每个分析页都有基于当前筛选结果的 AI 解读与对话；AI 只能提出"待确认"的筛选/指标建议，由你点击应用。
- 未配置模型时，仍提供明确标注的"本地规则建议（非 AI）"。
- 可通过"账户同步"把 AI 配置安全同步到你的网页账户（可撤销的一次性令牌）。

### 8. 报告导出与本地工作区

- 一键导出：**Excel 分析表**（多 Sheet 带口径说明）、**Word 经营报告**、**PDF 详细报告**（可选完整报告/仅当前章节/自定义范围）。
- 自动保存最近工作区（文件索引、字段口径、分析状态），重启自动恢复；可单独移除某份文件并重算其余，也可一键清空。

## 下载安装

Windows x64 安装包见 [Releases](../../releases)（NSIS 安装向导，已内嵌 Java 运行时，**无需预装 Java**）。安装后双击即用；程序只监听 `127.0.0.1`，不开放任何外部端口。

> 首次运行如遇 Windows SmartScreen"未知发布者"提示，点"仍要运行"即可（安装包暂未做代码签名）。

## 使用指南

1. **导入数据**：首页拖入或选择 Excel/CSV（可一次多选）。已有项目时再导入会询问"追加还是替换"。
2. **确认口径**：出现多套收入/成本候选列时，在"字段口径"卡中为收入、成本、日期等选定列，点"确认口径并重新分析"。
3. **看分析**：左侧切换五个视图；顶部可统一调整分析期间（累计/年度/月度，多月对比）。
4. **下钻探索**：表格行尾的 ↳ 按钮进入下一级维度；下钻条件以胶囊条显示，点 × 返回。筛选器支持多选与模糊搜索。
5. **配置 AI（可选）**：左下角"偏好设置"→ 选平台 → 选模型 → 填 API Key → 测试连接 → 保存。
6. **导出报告**：顶部工具栏导出 Excel 分析表、Word 经营报告；PDF 可勾选章节范围。

## 从源码构建

要求：**Java 17+、Maven 3.9+、Node.js 20+**。

```bash
# 构建后端（前端静态资源会一并打入 jar）
cd backend && mvn package

# 开发模式启动桌面应用
cd ../desktop && npm install && npm start
```

### 打包 Windows 安装包（macOS / Linux 上交叉构建）

```bash
# 1. 下载一份 Windows x64 JDK 17（只需其 jmods）
curl -L -o winjdk.zip "https://api.adoptium.net/v3/binary/latest/17/ga/windows/x64/jdk/hotspot/normal/eclipse"

# 2. 用本机 jlink 指向 Windows JDK 的 jmods，生成 Windows 运行时到 desktop/runtime-win-x64
cd desktop
"$JAVA_HOME/bin/jlink" \
  --module-path /path/to/winjdk/jmods \
  --add-modules java.base,java.compiler,java.desktop,java.instrument,java.logging,java.management,java.naming,java.net.http,java.prefs,java.rmi,java.scripting,java.security.jgss,java.security.sasl,java.sql,java.transaction.xa,java.xml,java.xml.crypto,jdk.crypto.ec,jdk.unsupported,jdk.zipfs \
  --strip-debug --no-header-files --no-man-pages --compress=2 \
  --output runtime-win-x64

# 3. 构建 NSIS 安装包（输出 desktop/dist/）
npm run dist:win
```

macOS 本机调试运行时可用 `desktop/build-runtime.sh` 生成；`npm run dist:mac` 打包 macOS 版本。

### 技术栈

- **desktop/**：Electron 桌面壳（拉起本地后端、原生菜单与快捷键、安全桥接、窗口状态恢复）
- **backend/**：Spring Boot 分析引擎与 API（Java 17；前端静态资源随 jar 打包）
- **frontend/**：无框架分析前端（原生 HTML/CSS/JS，SVG 图表，零依赖）

## 隐私与安全

- Excel 文件、经营数据、分析结果**只保存在本机**（`~/.datamaster/workspace`），不会上传任何云端。
- 本地后端仅监听 `127.0.0.1`；桌面壳与后端之间使用每次启动随机生成的令牌鉴权，渲染进程无任何 Node 能力。
- AI 解读只发送经营摘要与匿名列画像；同步令牌一次性、可撤销。
